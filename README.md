# java-project-tamplate
Javaの各種ライブラリを組み合わせたプロジェクトのテンプレート。  
各種ライブラリの使い方の勉強を主とする。

## 各種ライブラリ
- Spring-Boot
- Doma2
- Thymeleaf
- Lombok
- Gradle
- flyway
